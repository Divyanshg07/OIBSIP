# Divyansh Gupta Portfolio

A responsive, single-page personal portfolio for Divyansh Gupta, a frontend developer. It presents an introduction, technical skills, selected project placeholders, and contact details in a clean editorial layout.

## Features

- Responsive desktop and mobile layout
- Sticky navigation with smooth scrolling
- Hero section with developer profile placeholder
- About, skills, projects, and contact sections
- Project cards with GitHub link placeholders
- Google Fonts: Syne and DM Mono
- No build tools or JavaScript required

## Run locally

Open `index.html` in a modern web browser.


## Project structure

```text
index.html   Page content and structure
styles.css   Layout, palette, typography, and responsive styles
README.md    Project documentation
```

## Personalize

Update these values in `index.html` before publishing:

- Contact email: `divyansh.gupta@example.com`
- GitHub and LinkedIn placeholder URLs
- Project titles, descriptions, and repository links
- Avatar placeholder with a professional photo, if desired

## Technologies

- HTML5
- CSS3
- Google Fonts
