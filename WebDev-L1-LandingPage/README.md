# Divyansh Landing Page

A responsive, static personal-brand landing page built with HTML5 and CSS3.

## Features

- Sticky navigation with anchor links
- Hero section with headline and email call-to-action
- About, approach, quote, and contact sections
- Responsive Flexbox/Grid layout for mobile screens
- Consistent cream, green, coral, and charcoal palette
- No JavaScript or build tools required

## Run locally

Open `index.html` in any modern web browser.

## Project structure

```text
index.html   Page content and structure
style.css   Layout, typography, palette, and responsive styles
```

## Personalize

Update these placeholders in `index.html`:

- `Divyansh Gupta` — name and `Divyansh` — page title
- `divyan7499@gmail.com` — contact email
- Footer social-link `href` values
- Hero and section copy to match your work or brand

## Technologies

- HTML5
- CSS3
- Google Fonts: Manrope and DM Mono
